Load therapists from a PDF

How to get the PDF URL from Psych-Info
  1. Open psych-info.de
  2. Search for therapists at your preferred location
  3. Generate the PDF (button “Liste erstellen für Ausdruck”)
  4. Copy the URL of the opened PDF (e.g. https://psych-info.de/wp-content/uploads/pdf/psych-info_resultate_6a0ax0x0x0x0.pdf)
  5. Paste the URL into the field below

Paste the link to a Psych-Info "Resultate" PDF. We'll fetch and parse it on the server, then show the ranked list of therapists in the next step.

Have a local file? Upload a PDF / text file instead.
